# Datenschutz bei KitaPortal

## Grundprinzipien

- **Datensparsamkeit**: Wir speichern nur, was wir wirklich brauchen
- **Lokale Datenhaltung**: Alle Daten liegen auf dem Server der Kita – kein Drittanbieter hat Zugriff
- **Keine Weitergabe**: Keine Daten werden an Dritte verkauft oder weitergegeben
- **Transparenz**: Der gesamte Code ist öffentlich einsehbar

## Welche Daten werden gespeichert?

| Daten | Zweck | Speicherdauer |
|---|---|---|
| Name, E-Mail der Eltern | Login, Kommunikation | Bis Konto gelöscht wird |
| Name, Gruppe des Kindes | Zuordnung | Bis Kind die Kita verlässt |
| Krankmeldungen | Anwesenheitsplanung | 1 Jahr |
| Abholerlaubnisse | Sicherheit | 6 Monate |
| Allergien | Gesundheitsschutz | Bis Kind die Kita verlässt |
| Nachrichten | Kommunikation | 1 Jahr |

## DSGVO-Konformität

KitaPortal ist als Auftragsverarbeitungs-Tool konzipiert.
Die Kita ist datenschutzrechtlich Verantwortliche.

Ein Muster-Auftragsverarbeitungsvertrag (AVV) ist geplant.
