# Mitmachen bei KitaPortal

Danke, dass du mitmachen möchtest! KitaPortal lebt davon, dass Menschen wie du ihre Zeit einbringen.

## Wie fange ich an?

1. **Repo forken** – oben rechts auf "Fork" klicken
2. **Lokal klonen** – `git clone https://github.com/DEIN-USERNAME/kitaportal.git`
3. **Branch erstellen** – `git checkout -b feature/mein-feature`
4. **Änderungen committen** – `git commit -m "feat: kurze Beschreibung"`
5. **Pull Request erstellen** – zurück auf GitHub, "Compare & pull request"

## Was wird gebraucht?

Schau in die [offenen Issues](../../issues). Alles mit `good first issue` ist ein guter Einstieg.

## Code-Stil

- Python: [Black](https://black.readthedocs.io/) für Formatierung
- JavaScript/React: [Prettier](https://prettier.io/) + ESLint
- Commit-Messages: [Conventional Commits](https://www.conventionalcommits.org/) (`feat:`, `fix:`, `docs:`, ...)

## Fragen?

Einfach eine [Discussion](../../discussions) starten. Wir helfen gerne.
